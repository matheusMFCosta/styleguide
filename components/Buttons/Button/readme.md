Button Types

```js
<div>
  <Button className="mr2" style="primary">
    Primary
  </Button>
  <Button className="mr2" style="secondary">
    Secondary
  </Button>
  <Button isDisabled>Disabled</Button>
</div>
```

Button Sizes

```js
<div>
  <Button className="mr2" size="small">
    Small
  </Button>
  <Button className="mr2" size="normal">
    Normal
  </Button>
  <Button size="large">Large</Button>
</div>
```
